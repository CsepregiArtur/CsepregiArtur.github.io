Capabilities
Key Features
🔐

Single-Session Optimization

One authentication, all location queries in parallel, one logout. Eliminates N−1 redundant login cycles. Major v5.0 breakthrough.

⚡

Parallel Processing

ThreadPoolExecutor with configurable workers (tested at 10). 20 locations process in nearly the same time as 1.

📊

Dual Export Engine

xlsxwriter primary (formatting) + pandas fallback (robust) + CSV emergency (permission-safe). Never fails on Excel issues.

🔄

Retry Logic

3 retry attempts with configurable delay per location. Handles transient MES connection issues gracefully.

📋

FIFO/FEFO Sorting

Automated compliance sorting for regulated manufacturing. FIFO by storage box/index or FEFO by expiration date.

📦

Self-Contained .EXE

Single 31.3 MB executable. Zero Python or DLL dependencies for end users. Code-signed distribution.

Technology
Tech Stack
Python 3.12pythonnet (clr).NET DLL Interop ThreadPoolExecutorxlsxwriterpandas PyInstallerNewtonsoft.Json
Architecture: Python ↔ .NET interop via pythonnet for MES API access. Single-session singleton pattern for thread-safe parallel queries.

Export: Dual-engine Excel generation with 19-column standardized schema. FIFO/FEFO sorting for inventory compliance.

Distribution: PyInstaller single-file .exe (31.3 MB) with embedded Python runtime, .NET DLL, and configuration template. Code-signed for enterprise deployment.
