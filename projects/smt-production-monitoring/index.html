Capabilities
Key Features
🔄

Continuous Polling

SMB network share polling every 3 seconds across 30 XML data sources. Pattern matching with deduplication.

🌉

.NET Legacy Bridge

C# .NET 9 x86 service bridges Python 64-bit to legacy 32-bit DLL for master datasheet queries. HTTP on port 5001.

🛡️

Watchdog Monitor

Thread health monitoring, auto-restart of crashed orchestrator, 7-day rolling incident log. Self-healing architecture.

😴

Weekend Mode

Auto-pauses folder polling on Sat/Sun after 3h inactivity. Resumes Monday 07:00 automatically. Per-line independent.

💾

Crash Recovery

Full runtime state persisted to JSON every ~90 seconds. On restart, loads last saved state and resumes monitoring.

🧪

Simulation Mode

Built-in XML telemetry generator for offline testing, training, and demonstrations. 5 test cohorts for validation logic.

📊

Efficiency Analytics

Per-station efficiency charts, bottleneck detection, Pareto defect aggregation, and idle time tracking.

📋

CSV Export

Validation and telemetry reports in CSV format. Audit-ready documentation of station health and program compliance.

Technology
Tech Stack

Backend: Flask + Socket.IO with multi-threaded file polling, validation engine, and watchdog monitoring.

Frontend: React 18 + TypeScript + Vite + TailwindCSS + Lucide Icons. WebSocket real-time updates with auto-reconnect.

Bridge: C# .NET 9 x86 service bridging Python 64-bit to legacy 32-bit DLL for SAP datasheet queries.

Distribution: PyInstaller single-file .exe with obfuscated source, license SDK, and embedded frontend build.
